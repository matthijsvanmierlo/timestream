# Timestream

A modern, local-first timeline maker built with React and Tailwind CSS, called Timestream.

## Features
- **Privacy First**: All data is stored locally in your browser.
- **Project Management**: Create and manage multiple timeline projects.
- **Modern UI**: Clean, responsive interface with beautiful typography.
- **Unsplash Integration**: Easily find and add images to your events.
- **Customizable**: Choose from different themes, fonts, and colors for each timeline.
- **Sharing**: Share your timelines via URL (data is compressed into the link).

## Tech Stack
- React (via CDN)
- Tailwind CSS (via CDN)
- LZ-String (for URL compression)
- GitHub Pages ready

## Usage
Simply open `index.html` in your browser or serve it via a static file server.
